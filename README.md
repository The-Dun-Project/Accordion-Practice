# Accordion Practice

This repository tracks my 14-day practice project where I rebuild the same animated accordion component from scratch using HTML, CSS, and JavaScript.

## Goal
To improve:
- JavaScript logic
- DOM control
- interaction design
- consistency through repetition

## Project Features
- 3 accordion items
- smooth open and close
- rotating icon
- one item open at a time

## Structure
- `index.html` — structure
- `style.css` — styling and transitions
- `script.js` — interaction logic

## Practice Rule
I rebuild this project from scratch each day instead of endlessly editing one file.
